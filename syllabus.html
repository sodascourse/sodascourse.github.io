Prerequisite

  • Xcode 8.0 (or newer versions)
  • A Mac running macOS Sierra 10.12 or OS X El Capitan 10.11.5 (or newer updates of each OS versions)
    (laptops are recommended. you should bring your Mac to the class.)
  • An iOS device (recommended, not required, at least running iOS 10.)
  • This class is aimed for programming newbies, so there's no required class before this class.
